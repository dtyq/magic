Added Layout Variant Notes
Section Variant A — Left Dark Panel
Left 35% dark background (#2D2D2D) + sunburst decoration + chapter-badge + large title (white) + subtitle + three key points; right 65% SVG panoramic illustration scene + floating data cards. Best for section transitions that need generous illustration space.
Section Variant B — Full-Width Bottom Scene
Top text area (title + data card group) + full-width SVG horizontal panoramic scene at the bottom (foreground/midground/background: hills/buildings/people/plants), with strong visual impact. Best for nature, city, or ecology themed sections.
