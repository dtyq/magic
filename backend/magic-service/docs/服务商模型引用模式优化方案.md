# 服务商模型引用模式优化方案

## 1. 业务背景

### 1.1 现状描述

Magic Service 是一个基于多服务商的 AI 模型管理平台，支持多个组织使用各种 AI 服务提供商（如 OpenAI、Claude 等）的模型。

当前系统的工作机制：
- **官方组织**：作为模型的主要管理者，负责维护各个服务商的官方模型
- **业务组织**：通过同步机制获取官方组织的模型副本，形成自己的模型库
- **初始化流程**：新组织创建时，系统会从官方组织复制所有模型到该组织下

### 1.2 核心问题

随着平台规模的扩大，出现了严重的性能瓶颈：

#### 问题1：海量级联更新
- 假设官方组织有 **10万个模型**，同时有 **1000个业务组织**
- 当官方修改一个模型时，需要同步更新 **100万条记录**（10万 × 10个组织）
- 单次操作的性能开销极大，可能导致系统超时

#### 问题2：存储空间浪费
- 每个组织都保存完整的模型副本
- 大量重复数据占用存储空间
- 数据库容量快速膨胀

#### 问题3：数据一致性风险
- 同步过程中可能出现数据不一致
- 事务处理复杂度高
- 回滚机制困难

### 1.3 业务逻辑要求

系统设计了复杂的模型状态管理机制：

#### DisabledBy 字段逻辑
- `disabledBy = null`：正常激活状态
- `disabledBy = 'OFFICIAL'`：官方禁用，**用户完全不可见**
- `disabledBy = 'USER'`：用户禁用，**用户可见但处于禁用状态**

#### 关键业务规则
1. **官方禁用具有最高优先级**：一旦官方禁用模型，所有组织都看不到该模型
2. **用户禁用为个人选择**：用户可以禁用模型，但仍然可见，可以随时重新启用
3. **状态继承机制**：组织的模型状态需要综合考虑官方状态和用户自定义状态

```php
private function initMagicServiceProviderModels(string $organizationCode): bool
{
    $officialOrganization = config('service_provider.office_organization');
    if ($organizationCode === $officialOrganization) {
        return true;
    }
    
    // 获取官方组织下所有LLM类型服务商（排除Magic自身）
    $officeServiceProviders = $this->serviceProviderConfigRepository->findByConditions([
        'organization_code' => $officialOrganization,
        'category' => ServiceProviderCategory::LLM->value,
        'provider_code' => ['!=', ServiceProviderCode::Magic->value],
    ]);
    
    // 批量复制官方模型到新组织的Magic服务商下
    $modelsToSave = [];
    foreach ($officeServiceProviders as $officeProvider) {
        $officeModels = $this->getModelsByServiceProviderConfigId($officeProvider->getId());
        
        foreach ($officeModels as $baseModel) {
            $newModel = clone $baseModel;
            $newModel->setId(null);
            $newModel->setOrganizationCode($organizationCode);
            $newModel->setServiceProviderConfigId($magicServiceProviderConfigId);
            $newModel->setModelParentId($baseModel->getId()); // 设置父级关系
            $newModel->setIsOffice(true); // 标记为官方模型
            
            // 状态继承：服务商激活 AND 模型激活 = 新模型激活
            $bothActive = ($baseModel->getStatus() === Status::ACTIVE->value)
                && ($officeConfigMap[$baseModel->getServiceProviderConfigId()]['status'] === Status::ACTIVE->value);
            $newModel->setStatus($bothActive ? Status::ACTIVE->value : Status::DISABLE->value);
            
            $modelsToSave[] = $newModel;
        }
    }
    
    // 批量保存
    $this->batchSaveModels($modelsToSave);
    return true;
}
```

#### updateModelStatus - 模型状态更新

```php
public function updateModelStatus(string $modelId, Status $status, string $organizationCode): void
{
    $serviceProviderModelsEntity = $this->getModelByIdAndOrganizationCode($modelId, $organizationCode);
    
    // 设置禁用来源
    $disabledBy = $this->isOfficial($organizationCode) ? DisabledByType::OFFICIAL : DisabledByType::USER;
    
    // 如果是官方组织操作，同步更新所有相关模型
    if ($this->isOfficial($organizationCode)) {
        $serviceProviderCategory = ServiceProviderCategory::from($serviceProviderModelsEntity->getCategory());
        
        if ($serviceProviderCategory === ServiceProviderCategory::VLM) {
            $this->syncUpdateModelsStatusByVLM($serviceProviderModelsEntity->getModelVersion(), $status, $disabledBy);
        } elseif ($serviceProviderCategory === ServiceProviderCategory::LLM) {
            $this->syncUpdateModelsStatusByLLM((int) $serviceProviderModelsEntity->getId(), $status, $disabledBy);
        }
    }
    
    // 更新当前模型
    $this->serviceProviderModelsRepository->updateModelStatusAndDisabledBy($modelId, $organizationCode, $status, $disabledBy);
}
```

### 2.4 查询过滤逻辑

#### buildServiceProviderConfigDTO - 模型过滤
```php
private function buildServiceProviderConfigDTO(array $serviceProviderConfigs, array $models): array
{
    foreach ($serviceProviderConfigs as $config) {
        $configModels = array_filter($models, function($model) use ($config) {
            return $model->getServiceProviderConfigId() === $config->getId();
        });
        
        // 官方服务商的特殊过滤逻辑
        if ($this->isOfficialServiceProvider($config)) {
            $filteredModels = array_filter($configModels, function ($model) {
                return $model->getStatus() === Status::ACTIVE->value
                    || ($model->getStatus() === Status::DISABLE->value && $model->getDisabledBy() === DisabledByType::USER->value);
            });
            $config->setModels(array_values($filteredModels));
        } else {
            $config->setModels($configModels);
        }
    }
    
    return $serviceProviderConfigs;
}
```

**关键过滤规则**：
- 激活模型：直接显示
- 用户禁用模型：显示但标记为禁用状态，用户可重新启用
- 官方禁用模型：**完全不显示**，用户无法看到

## 3. 优化方案设计

### 3.1 核心设计思想

#### 引用模式替代复制模式
- **当前**：每个组织保存完整的模型副本
- **优化后**：组织只保存与官方模型的差异化配置

#### 按需生成原则
- **当前**：预先同步所有模型
- **优化后**：只有当组织修改模型状态时才生成记录

#### 查询时合并策略
- **当前**：直接查询组织模型
- **优化后**：查询官方模型 + 组织自定义配置，运行时合并

### 4.2 Repository 层实现

#### OrganizationModelOverrideRepository
```php
<?php

declare(strict_types=1);

namespace App\Domain\ModelAdmin\Repository\Persistence;

use App\Domain\ModelAdmin\Entity\OrganizationModelOverrideEntity;
use App\Domain\ModelAdmin\Repository\OrganizationModelOverrideRepositoryInterface;
use App\Infrastructure\Repository\AbstractRepository;

class OrganizationModelOverrideRepository extends AbstractRepository implements OrganizationModelOverrideRepositoryInterface
{
    protected string $entity = OrganizationModelOverrideEntity::class;

    protected string $table = 'organization_model_overrides';

    /**
     * 根据组织编码获取所有覆写配置
     */
    public function findByOrganizationCode(string $organizationCode): array
    {
        return $this->findBy(['organization_code' => $organizationCode]);
    }

    /**
     * 根据组织编码和官方模型ID获取覆写配置
     */
    public function findByOrganizationAndOfficialModel(string $organizationCode, int $officialModelId): ?OrganizationModelOverrideEntity
    {
        return $this->findOneBy([
            'organization_code' => $organizationCode,
            'official_model_id' => $officialModelId,
        ]);
    }

    /**
     * 批量获取组织的覆写配置
     */
    public function findByOrganizationAndOfficialModels(string $organizationCode, array $officialModelIds): array
    {
        if (empty($officialModelIds)) {
            return [];
        }

        $qb = $this->createQueryBuilder('o');
        return $qb->where('o.organization_code = :orgCode')
            ->andWhere($qb->expr()->in('o.official_model_id', ':modelIds'))
            ->setParameter('orgCode', $organizationCode)
            ->setParameter('modelIds', $officialModelIds)
            ->getQuery()
            ->getResult();
    }

    /**
     * 创建或更新覆写配置
     */
    public function createOrUpdate(OrganizationModelOverrideEntity $override): OrganizationModelOverrideEntity
    {
        $existing = $this->findByOrganizationAndOfficialModel(
            $override->getOrganizationCode(),
            $override->getOfficialModelId()
        );

        if ($existing) {
            // 更新现有记录
            $this->updateOverrideFields($existing, $override);
            return $this->save($existing);
        } else {
            // 创建新记录
            return $this->save($override);
        }
    }

    /**
     * 删除覆写配置
     */
    public function deleteByOrganizationAndOfficialModel(string $organizationCode, int $officialModelId): bool
    {
        $override = $this->findByOrganizationAndOfficialModel($organizationCode, $officialModelId);
        if ($override) {
            $this->delete($override);
            return true;
        }
        return false;
    }

    /**
     * 批量删除组织的所有覆写配置
     */
    public function deleteByOrganizationCode(string $organizationCode): int
    {
        $qb = $this->createQueryBuilder('o');
        return $qb->delete()
            ->where('o.organization_code = :orgCode')
            ->setParameter('orgCode', $organizationCode)
            ->getQuery()
            ->execute();
```

```php
    /**
     * 获取组织的模型列表（合并官方模型和组织自定义）
     */
    public function getModelsForOrganization(
        string $organizationCode, 
        ?ServiceProviderCategory $category = null
    ): array {
        // 1. 获取所有官方激活模型
        $officialModels = $this->officialModelRepository->findOfficialActiveModels($category);
        
        if (empty($officialModels)) {
            return [];
        }
        
        // 2. 过滤对组织可见的模型
        $visibleOfficialModels = array_filter($officialModels, function($model) use ($organizationCode) {
            return $this->officialModelRepository->isModelVisibleToOrganization($model->getId(), $organizationCode);
        });
        
        if (empty($visibleOfficialModels)) {
            return [];
        }
        
        // 3. 获取组织的覆写配置
        $officialModelIds = array_map(fn($model) => $model->getId(), $visibleOfficialModels);
        $overrides = $this->overrideRepository->findByOrganizationAndOfficialModels($organizationCode, $officialModelIds);
        
        // 4. 合并官方模型和组织覆写
        return $this->mergeOfficialModelsWithOverrides($visibleOfficialModels, $overrides);
    }

    /**
     * 获取单个模型的完整信息
     */
    public function getModelById(string $modelId, string $organizationCode): ?ServiceProviderModelsEntity
    {
        // 1. 尝试通过 model_id 找到官方模型
        $officialModel = $this->officialModelRepository->findOfficialModelByModelId($modelId);
        if (!$officialModel) {
            return null;
        }
        
        // 2. 检查模型对组织是否可见
        if (!$this->officialModelRepository->isModelVisibleToOrganization($officialModel->getId(), $organizationCode)) {
            return null;
        }
        
        // 3. 获取组织的覆写配置
        $override = $this->overrideRepository->findByOrganizationAndOfficialModel($organizationCode, $officialModel->getId());
        
        // 4. 合并配置
        return $this->mergeOfficialModelWithOverride($officialModel, $override, $organizationCode);
    }

    /**
     * 检查模型对组织是否可见
     */
    public function isModelVisibleToOrganization(int $officialModelId, string $organizationCode): bool
    {
        return $this->officialModelRepository->isModelVisibleToOrganization($officialModelId, $organizationCode);
    }

    /**
     * 合并官方模型和组织覆写配置
     */
    private function mergeOfficialModelsWithOverrides(array $officialModels, array $overrides): array
    {
        $overrideMap = [];
        foreach ($overrides as $override) {
            $overrideMap[$override->getOfficialModelId()] = $override;
        }
        
        $result = [];
        foreach ($officialModels as $officialModel) {
            $override = $overrideMap[$officialModel->getId()] ?? null;
            $merged = $this->mergeOfficialModelWithOverride($officialModel, $override);
            
            // 只返回非官方禁用的模型
            if ($merged && $merged->getDisabledBy() !== DisabledByType::OFFICIAL->value) {
                $result[] = $merged;
            }
        }
        
        return $result;
    }

    /**
     * 合并单个官方模型和覆写配置
     */
    private function mergeOfficialModelWithOverride(
        ServiceProviderModelsEntity $officialModel, 
        ?OrganizationModelOverrideEntity $override,
        ?string $organizationCode = null
    ): ?ServiceProviderModelsEntity {
        
        // 检查官方是否禁用了该模型
        if ($officialModel->getDisabledBy() === DisabledByType::OFFICIAL->value) {
            return null; // 官方禁用的模型完全不可见
        }
        
        // 创建合并后的模型实体
        $merged = clone $officialModel;
        
        if ($override) {
            // 应用组织的覆写配置
            $merged->setId($override->getId()); // 使用覆写记录的ID
            $merged->setOrganizationCode($override->getOrganizationCode());
            $merged->setServiceProviderConfigId($override->getServiceProviderConfigId());
            $merged->setModelParentId($officialModel->getId()); // 设置父级关系
            
            // 覆写字段（如果覆写中有值）
            if ($override->getName() !== null) {
                $merged->setName($override->getName());
            }
            if ($override->getIcon() !== null) {
                $merged->setIcon($override->getIcon());
            }
            if ($override->getConfig() !== null) {
                $merged->setConfig($override->getConfig());
            }
            if ($override->getDescription() !== null) {
                $merged->setDescription($override->getDescription());
            }
            if ($override->getSort() !== null) {
                $merged->setSort($override->getSort());
            }
            if ($override->getTranslate() !== null) {
                $merged->setTranslate($override->getTranslate());
            }
            
            // 状态字段使用覆写的值
            $merged->setStatus($override->getStatus());
            $merged->setDisabledBy($override->getDisabledBy());
            $merged->setSuperMagicDisplayState($override->getSuperMagicDisplayState());
            
            // 时间字段
            $merged->setUpdatedAt($override->getUpdatedAt());
            
        } else {
            // 没有覆写，但需要调整组织信息
            if ($organizationCode) {
                $merged->setOrganizationCode($organizationCode);
                $merged->setModelParentId($officialModel->getId());
            }
        }
        
        // 设置为官方模型标识
        $merged->setIsOffice(true);
        
        return $merged;
    }
```

```php
            $serviceProviderConfigDTOS = array_filter($serviceProviderConfigDTOS, function ($serviceProviderConfigDTO) {
                return ServiceProviderCode::from($serviceProviderConfigDTO->getProviderCode()) !== ServiceProviderCode::Magic;
            });
            $serviceProviderConfigDTOS = array_values($serviceProviderConfigDTOS);
        }
        
        return $serviceProviderConfigDTOS;
    }

    /**
     * 更新模型状态 - 使用新的覆写服务
     */
    public function updateModelStatus(string $modelId, int $status, string $organizationCode): void
    {
        // 1. 通过 model_id 找到官方模型
        $officialModel = $this->modelQueryService->findOfficialModelByModelId($modelId);
        if (!$officialModel) {
            ExceptionBuilder::throw(ServiceProviderErrorCode::ModelNotFound, '模型不存在');
        }
        
        // 2. 检查是否为官方组织操作
        $officeOrganization = config('service_provider.office_organization');
        if ($organizationCode === $officeOrganization) {
            // 官方组织直接更新官方模型，会影响所有组织
            $this->serviceProviderDomainService->updateOfficialModelStatus($officialModel->getId(), Status::from($status));
        } else {
            // 非官方组织创建或更新覆写配置
            $serviceProviderConfigId = $this->getOrganizationServiceProviderConfigId($organizationCode, $officialModel);
            $this->overrideService->updateModelStatus(
                $organizationCode,
                $officialModel->getId(),
                Status::from($status),
                $serviceProviderConfigId
            );
        }
    }

    /**
     * 构建服务商配置DTO列表，包含模型信息
     */
    private function buildServiceProviderConfigDTOsWithModels(array $serviceProviderConfigs, array $models): array
    {
        // 将模型按服务商配置ID分组
        $modelsByConfigId = [];
        foreach ($models as $model) {
            $configId = $model->getServiceProviderConfigId();
            if (!isset($modelsByConfigId[$configId])) {
                $modelsByConfigId[$configId] = [];
            }
            $modelsByConfigId[$configId][] = $model;
        }
        
        // 为每个服务商配置设置对应的模型列表
        $result = [];
        foreach ($serviceProviderConfigs as $config) {
            $configModels = $modelsByConfigId[$config->getId()] ?? [];
            
            // 应用官方服务商的特殊过滤逻辑
            if ($this->isOfficialServiceProvider($config)) {
                $filteredModels = array_filter($configModels, function ($model) {
                    return $model->getStatus() === Status::ACTIVE->value
                        || ($model->getStatus() === Status::DISABLE->value && $model->getDisabledBy() === DisabledByType::USER->value);
                });
                $configModels = array_values($filteredModels);
            }
            
            $configDTO = new ServiceProviderConfigDTO($config->toArray());
            $configDTO->setModels($configModels);
            $result[] = $configDTO;
        }
        
        return $result;
    }

    /**
     * 获取组织的服务商配置ID
     */
    private function getOrganizationServiceProviderConfigId(string $organizationCode, ServiceProviderModelsEntity $officialModel): int
    {
        // 根据官方模型的类别，找到组织对应的服务商配置ID
        $category = ServiceProviderCategory::from($officialModel->getCategory());
        
        if ($category === ServiceProviderCategory::LLM) {

            return $this->serviceProviderDomainService->getMagicServiceProviderConfigId($organizationCode);
        } else {
            // VLM 等其他类型，需要找到对应的服务商配置
            return $this->serviceProviderDomainService->getCorrespondingServiceProviderConfigId(
                $organizationCode,
                $officialModel->getServiceProviderConfigId()
            );
        }
```

```php
        $this->info("- 总模型数: {$totalModels}");
        $this->info("- 官方模型数: {$officialModels}");
        $this->info("- 组织模型数: {$organizationModels}");
        
        // 2. 获取所有非官方组织的模型
        $nonOfficialModels = $this->modelsRepository->findNonOfficialModels();
        
        $this->info("需要迁移的组织模型: " . count($nonOfficialModels));
        
        if ($dryRun) {
            $this->showMigrationPreview($nonOfficialModels);
            return;
        }
        
        // 3. 执行迁移
        $this->performMigration($nonOfficialModels);
        
        $this->info('迁移完成!');
    }

    private function showMigrationPreview(array $models): void
    {
        $this->info("\n=== 迁移预览 ===");
        
        $groupedByOrg = [];
        foreach ($models as $model) {
            $orgCode = $model->getOrganizationCode();
            if (!isset($groupedByOrg[$orgCode])) {
                $groupedByOrg[$orgCode] = [];
            }
            $groupedByOrg[$orgCode][] = $model;
        }
        
        foreach ($groupedByOrg as $orgCode => $orgModels) {
            $this->info("\n组织: {$orgCode}");
            $this->info("  模型数量: " . count($orgModels));
            
            // 统计需要创建覆写的模型
            $needOverrideCount = 0;
            foreach ($orgModels as $model) {
                if ($this->needsOverride($model)) {
                    $needOverrideCount++;
                }
            }
            
            $this->info("  需要创建覆写: {$needOverrideCount}");
            $this->info("  可以删除: " . (count($orgModels) - $needOverrideCount));
        }
    }

    private function performMigration(array $models): void
    {
        $this->info("\n开始执行迁移...");
        
        $createdOverrides = 0;
        $deletedModels = 0;
        $errors = 0;
        
        $progressBar = $this->output->createProgressBar(count($models));
        $progressBar->start();
        
        foreach ($models as $model) {
            try {
                if ($this->needsOverride($model)) {
                    // 创建覆写记录
                    $override = $this->createOverrideFromModel($model);
                    $this->overrideRepository->save($override);
                    $createdOverrides++;
                }
                
                // 删除原有组织模型
                $this->modelsRepository->delete($model);
                $deletedModels++;
                
            } catch (\Exception $e) {
                $this->error("迁移模型 {$model->getId()} 失败: " . $e->getMessage());
                $errors++;
            }
            
            $progressBar->advance();
        }
        
        $progressBar->finish();
        
        $this->info("\n\n=== 迁移结果 ===");
        $this->info("创建覆写记录: {$createdOverrides}");
        $this->info("删除组织模型: {$deletedModels}");
        $this->info("错误数量: {$errors}");
    }

    /**
     * 判断是否需要创建覆写记录
     */
    private function needsOverride($model): bool
    {
        // 如果模型有以下任一情况，需要创建覆写：
        // 1. 状态为禁用且为用户禁用
        // 2. 有自定义的名称、图标、配置等
        // 3. super_magic_display_state 不为默认值
        
        if ($model->getStatus() === Status::DISABLE->value && $model->getDisabledBy() === DisabledByType::USER->value) {
            return true;
        }
        
        if ($model->getSuperMagicDisplayState() !== 0) {
            return true;
        }
        
        // 检查是否有自定义字段（与官方模型对比）
        $officialModel = $this->getOfficialModelByParentId($model->getModelParentId());
        if (!$officialModel) {
            return true; // 找不到官方模型，保险起见创建覆写
        }
        
        // 比较字段差异
        if ($model->getName() !== $officialModel->getName() ||
            $model->getIcon() !== $officialModel->getIcon() ||
            $model->getDescription() !== $officialModel->getDescription() ||
            $model->getSort() !== $officialModel->getSort()) {
            return true;
        }
        
        return false;
    }

    /**
     * 从组织模型创建覆写记录
     */
    private function createOverrideFromModel($model): OrganizationModelOverrideEntity
    {
        $override = new OrganizationModelOverrideEntity();
        $override->setOrganizationCode($model->getOrganizationCode());
        $override->setOfficialModelId($model->getModelParentId());
        $override->setServiceProviderConfigId($model->getServiceProviderConfigId());
        
        // 获取官方模型用于对比
        $officialModel = $this->getOfficialModelByParentId($model->getModelParentId());
        
        // 只设置与官方模型不同的字段
        if ($officialModel && $model->getName() !== $officialModel->getName()) {
            $override->setName($model->getName());
        }
        
        if ($officialModel && $model->getIcon() !== $officialModel->getIcon()) {
            $override->setIcon($model->getIcon());
        }
        
        if ($officialModel && $model->getDescription() !== $officialModel->getDescription()) {
            $override->setDescription($model->getDescription());
        }
        
        if ($officialModel && $model->getSort() !== $officialModel->getSort()) {
            $override->setSort($model->getSort());
        }
        
        // 配置和翻译字段比较复杂，暂时都保存
        $override->setConfig($model->getConfig());
        $override->setTranslate($model->getTranslate());
        
        // 状态字段
        $override->setStatus($model->getStatus());
        $override->setDisabledBy($model->getDisabledBy());
        $override->setSuperMagicDisplayState($model->getSuperMagicDisplayState());
        
        return $override;
    }

    /**
     * 根据父级ID获取官方模型
     */
    private function getOfficialModelByParentId(int $parentId)
    {
        if ($parentId <= 0) {
            return null;
        }
        
        return $this->modelsRepository->find($parentId);
    }
}
```
