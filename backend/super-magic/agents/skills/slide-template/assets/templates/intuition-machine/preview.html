OVERVIEW
本研究探讨大型语言模型在多任务场景下的泛化能力, Highlight分析指令微调与上下文学习的协同效应.
TECHNICAL
模型基于 Transformer 解码器架构, 采用 Flash Attention 2.0 优化显存占用, 支持 32K 上下文长度.
RESULTS
在 MMLU, HumanEval, GSM8K 三项基准测试中均超越同参数量基线模型, 平均提升 7.3%.
Accordion · Accordion
ARCHITECTURE ▼
采用标准 Transformer 解码器架构, 包含 32 层自注意力模块, 隐藏层维度 4096, 注意力头数 32, 前馈层维度 16384.
TRAINING ▼
在 2T tokens 语料上预训练, 使用 AdamW 优化器, 学习率 3e-4, 余弦退火调度, 全局批次大小 4M tokens.
ALIGNMENT ▼
结合监督微调(SFT)与基于人类反馈的强化学习(RLHF), 采用 PPO 算法优化奖励模型目标.
EVALUATION ▼
在 MMLU, BBH, HumanEval, GSM8K, MATH 等 12 项标准基准上进行全面评测, 结果均达到同规模 SOTA 水平.
Tooltip & Annotation
模型使用 自注意力机制 SELF-ATTENTION · 自注意力
计算序列内各 token 的相关性权重 对输入进行编码, 并通过 残差连接 RESIDUAL CONNECTION
缓解梯度消失, 加速深层网络训练 稳定训练过程.

# Transformer attention (simplified) def attention(Q, K, V, d_k): scores = Q @ K.T / d_k**0.5 weights = softmax(scores) return weights @ V
使用 scaled dot-product 计算注意力权重, 缩放因子 √d_k 防止点积值过大导致梯度消失.
