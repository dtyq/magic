[🦞 OpenClaw](https://github.com/openclaw/openclaw) は優れた個人向けAIアシスタントです。主要なIMすべてを会話チャネルとして接続し、あらゆるLLMに対応し、24時間365日自律的に稼働します。

しかし、エンタープライズの文脈に持ち込むと、新たな課題が自然と浮上します。個人アカウントに散在するデータ、予算の管理機能がない、プレーンテキストに留まるアウトプット、承認ゲートのないハイリスクな操作。

Magicはまさにこうした課題のために構築されました。**セキュリティ、制御、直接的なビジネス成果、そして24時間365日の自律運用**を実現するエンタープライズAIエージェントプラットフォームです。

---

## 試行錯誤はもう終わり。エンタープライズAIエンジンを構築しよう。

個人向けAIツールは、大規模に展開すると常に同じ壁にぶつかります。Magicはそれぞれの課題に対処します：

- 従業員のアカウントにデータがロックされ、退職とともに消失 → 統合データハブにより、組織のナレッジは組織に残る
- 予測不能なAPIコスト、月末の予算超過 → 部門別・ユーザー別・タスク別の予算上限設定
- スタッフがサードパーティツールを使用し、コアデータが危険に → 社内サンドボックス分離でデータは信頼境界の外に出ない
- AIオートメーションがファイルを削除したり、誤ったメールを送信 → ハイリスク操作には人間の承認が必要、人間が制御権を保持
- ERP、CRMに接続できない → 社内システムをデジタル従業員としてラップし、全員がアクセス可能に
- 出力がテキストだけで、手動でスライドにフォーマットする必要がある → 完成した成果物を直接提供：ダッシュボード、レポート、PPT、Excelファイル

## あらゆる規模に対応

Magicは大企業だけのものではありません。**1人の事業者から1万人の組織まで、同じ問題を解決します：1人分のコストで100人分のアウトプット。**

**OPC（ワンパーソンカンパニー）** や **OPT（ワンパーソンチーム）** の運用モデルに自然にフィットします。少ない資産、ゼロの人件費オーバーヘッド、具体的な成果の迅速な提供、必要に応じて新しい機能をプラグイン。ソロ創業者でも小規模チームでも、AIワークフォース全体を指揮できます。

**ソロ創業者 / 1人チーム**

まだ採用していない役割をAIにカバーさせましょう — マーケティング、オペレーション、法務、データ分析、カスタマーサポート、デザイン、コピーライティング、財務。人件費ゼロ、24時間365日のアウトプット。ダッシュボード、レポート、契約書ドラフトを迅速に作成。新しい機能が必要？エージェントを追加するだけです。

**エンタープライズ / 中規模チーム**

組織全体でAIを管理する集中型プラットフォームにより、データ漏洩リスクを排除。部門予算は透明で追跡可能。社内の専門知識はデジタル従業員としてパッケージ化され、部門間で再利用可能 — ナレッジが組織から流出することはありません。ハイリスク操作には承認が必要。サンドボックス分離によりデータ漏洩を防止します。

## 7つのエンタープライズグレードのコア機能

### 1. エンタープライズナレッジの統合

断片化された社内システム（ERP / CRM / データベース）やドメイン専門知識を、**全員がアクセスできるデジタル従業員**に深くカプセル化。サイロを解体し、散在するAPI呼び出しを大規模に再利用可能なコアデジタルアセットに変換します。

![digital-employee-market](https://public-cdn.magicrew.ai/static/img/3.0/digital-employee-market.png)

### 2. 成果物としてのアウトプット

「チャット」で終わりではありません。組み込みのレンダリングフレームワークが、AIの出力を完成した成果物に直接変換します — **PPT、データダッシュボード、議事録、プロフェッショナルレポート、Excelファイル、無限キャンバス（画像作成）** — 後処理不要で、すぐに使えます。

![solution-ppt-demo](https://public-cdn.magicrew.ai/static/img/3.0/solution-ppt-demo.png)
![earnings-call-analysis](https://public-cdn.magicrew.ai/static/img/3.0/earnings-call-analysis.png)
![canvas-poster-design](https://public-cdn.magicrew.ai/static/img/3.0/canvas-poster-design.png)

### 3. エンタープライズセキュリティ＆コンプライアンス

すべてのエージェントが**独自のサンドボックスコンテナ**内で実行され、メインシステムから分離された別のVPCでプライベートエンドポイント経由で接続 — 不正アクセスのリスクはゼロです。Sidecarネットワークプロキシがユーザーごとにトラフィックを独立して管理し、**テナント間でリソースとデータの完全な分離**を実現。厳格な**プラグインセキュリティレビュー**が公開前に悪意のあるコードを検出し、エンタープライズデータを常に信頼境界内に保ちます。

### 4. ヒューマン・イン・ザ・ループ制御

エージェントがハイリスクな操作を試みると、承認ワークフローがトリガーされます。定型的なアクションは自律的に実行されますが、データの削除やメールの送信などの破壊的な操作は**明示的な人間の確認が必要**です。意思決定権は常に人間にあります。

### 5. きめ細かなコスト管理

エンタープライズグレードのコストコンパスにより、**部門別、ユーザー別、エージェント別の日次予算**を精密に設定できます。AIの支出が予測可能かつ制御可能になり、すべての費用が正当化されます。

### 6. チーム全体のコラボレーション

複数のメンバーが1つのプロジェクトを共有し、それぞれが異なるモジュールを担当し、リアルタイムの可視性を持ちながら並行して作業を進めます。エキスパートユーザーは同僚のプロジェクトに参加して、その場でサポートできます。**経験はプロジェクトを通じて自然に蓄積され、チーム全体で複利的に成長します。** 進捗はWeChat Work、DingTalk、Larkグループに自動レポートでき、真のゼロフリクションコラボレーションを実現します。

### 7. オープンエコシステム互換性

**[Anthropic Skills](https://docs.anthropic.com/en/docs/agents-and-tools/tool-use/overview) エコシステム**および **[OpenClaw Skills](https://github.com/openclaw/openclaw) エコシステム**と完全互換。既存のツールやスキルをそのまま接続でき、エンタープライズグレードへの移行コストはゼロです。

## パーソナルAIアシスタント + エキスパートエージェント：全員がAI軍団を指揮する

Magicはエンタープライズに2つの補完的な機能を提供します：

**すべての従業員にパーソナルAIアシスタント**

各従業員に専属の🦞を割り当てるようなものです。24時間365日対応可能。パーソナルアシスタントはIMに接続するだけではありません — **すべてに接続**します：カレンダー、メール、社内システム、データ、ツール、そしてスペシャリストエージェント。1つの指示で適切なリソースと専門知識を動員 — これが「1人分のコストで100人分のアウトプット」の真の意味です。

**エキスパートエージェントはドメインスペシャリスト**

法務、財務、営業、オペレーション、その他あらゆる機能のノウハウとワークフローをデジタル従業員にコード化。各エキスパートエージェントは**そのドメイン内で深く包括的**であり、ユーザーがパーソナルアシスタントを通じて呼び出すことができます。新入社員がエキスパートを呼び出せば、初日からシニアレベルの判断力を得られます。ドメインの専門知識は個人に依存するものではなくなり、再利用可能な組織の資産となります。

**意思決定者の方へ、以下のシナリオを想像してください：**

### 8人で80人分の仕事

> 越境EC企業が8人のチームで運営。各メンバーにはパーソナルAIアシスタントがあり、商品選定、出品、広告、カスタマーサービス、物流、翻訳のエキスパートエージェントと接続 — エンドツーエンドで完全自動化。競合他社は同じ仕事を80人で行っています。しかもより遅い。

### 月次レポートではなく、いつでもリアルタイムの経営実態

> 午前2時、ふと疑問が浮かぶ：「東日本の粗利率は今四半期持ちこたえられるか？」パーソナルアシスタントがERP、財務システム、CRMに接続し、財務エキスパートエージェントを呼び出し、30秒でライブダッシュボードを提供。経営判断のスピードが、もはやレポートサイクルに縛られることはありません。

### グローバルビジネス、オフラインなし

> ニューヨークのクライアントが午前3時に緊急メールを送信。カスタマーサービスのエキスパートエージェントがすでに対応中。製品を理解し、返品ポリシーを把握し、自然な英語でコミュニケーション。朝を迎えたとき、目にするのはTo-Doリストではなく、解決済みチケットのサマリーです。タイムゾーンはもはやボトルネックではありません。

### 発生する前にリスクを遮断

> すべての送信契約が、クライアントに届く前に法務エキスパートエージェントを通過。リスクのある条項はフラグが付けられ、修正案が提案されます。必要に応じて人間の承認がトリガーされます。コンプライアンスは速度低下の原因ではなく、バックグラウンドで静かに稼働するセーフティネットです。

### 組織の記憶が複利で成長する価値

> すべてのビジネス判断、解決された問題、クライアントとのやり取りが、エキスパートエージェントをよりスマートにします。1年後、AIワークフォースは1人の経験ではなく、何百人もの集合知を持っています — そして休暇を取ることも、退職することも、知識を出し惜しみすることもありません。

### 新入社員。初日。シニアレベルのアウトプット。

> 以前は、優秀なプロジェクトマネージャーを育てるのに6か月かかりました。今では初日に、パーソナルアシスタントが新入社員をプロジェクト管理エキスパートエージェント、業界ナレッジベース、過去の事例ライブラリに接続。すべての落とし穴はすでに文書化され、すべてのテンプレートはすでにパッケージ化されています。立ち上げ期間は6か月から1週間に短縮され、初日からフル生産性を発揮します。

### 3人で10市場

> 3人のグローバル展開チームが10か国に参入する必要があります。パーソナルアシスタントが市場拡大エキスパートエージェントを呼び出し、フルプレイブックを実行：現地規制の調査、準拠した製品説明とマーケティングコピーの生成、現地プラットフォームでの出品管理、地域をまたいだ注文とアフターサービスの追跡。翻訳された数段落ではなく、エンドツーエンドの完全な市場参入オペレーションです。

### 退職するベテランの知識が組織から失われない

> アフターサービスチームには、顧客が症状を説明した瞬間に故障を診断できるベテランエンジニアがいます — 何を確認し、どう修理し、部品をどこで調達するか。その専門知識はすべてアフターサービスエキスパートエージェントにエンコードされました：症状 → 診断パス → 解決策 → 部品調達、完全な意思決定チェーン。若手技術者が難しい問題に直面し、パーソナルアシスタントに問い合わせると、ベテランレベルの診断が得られます。

### 結論の出ない2時間の会議はもう不要

> 会議前：パーソナルアシスタントが関連データと未解決事項を自動コンパイル。会議中：会議エキスパートエージェントがリアルタイムで書き起こし、議論のポイントをフラグし、すべてのアクションアイテムを追跡。終了5分後：担当者と期限付きの構造化されたサマリーが全参加者にプッシュされます。会議は本来あるべき姿になります：意思決定の場であり、時間の浪費ではなく。

## 🚀 クイックスタート

### セルフホストデプロイ

**要件：** Docker + curl。macOSおよびLinuxに対応（Windowsサポートは近日対応予定）。

```bash
curl -fsSL https://getmagicrew.sh | bash
```

スクリプトがすべてを処理します — クラスター作成、インフラストラクチャ、サービスデプロイ。完了後：

- Web UI: **http://localhost:38080**
- 停止: `magicrew teardown`

→ [デプロイガイド（詳細）](./docs/en/development/deploy/docker.md)

### クラウドサービス

セルフホストしたくない場合は、クラウド版をご利用ください — サインアップするだけで、設定不要：

- **中国**: [Magic](https://www.letsmagic.cn)
- **海外**: [MagiCrew](https://www.magicrew.ai)

### エンタープライズ版

チームやエンタープライズ向けに、プライベートデプロイ、専用モデル統合、社内システムとの深いカスタム連携など、強化された管理機能を提供します。ニーズについては[メールでお問い合わせ](mailto:bd@dtyq.com?subject=[GitHub]Business%20License%20Inquiry)ください。
