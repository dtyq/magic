# 服务商模型引用模式优化方案（简化版）

## 1. 业务背景

### 1.1 现状描述

Magic Service 是一个基于多服务商的 AI 模型管理平台，支持多个组织使用各种 AI 服务提供商（如 OpenAI、Claude 等）的模型。

当前系统的工作机制：
- **官方组织**：维护各个服务商的官方模型（OpenAI、Claude等），**没有Magic服务商**
- **业务组织**：通过同步机制获取官方组织的模型副本，放在Magic服务商下
- **初始化流程**：新组织创建时，系统会从官方组织复制所有模型到该组织的Magic服务商下

### 1.2 核心问题

随着平台规模的扩大，出现了严重的性能瓶颈：

#### 问题1：海量级联更新
- 假设官方组织有 **10万个模型**，同时有 **1000个业务组织**
- 当官方修改一个模型时，需要同步更新 **100万条记录**（10万 × 1000个组织）
- 单次操作的性能开销极大，可能导致系统超时

#### 问题2：存储空间浪费
- 每个组织都保存完整的模型副本
- 大量重复数据占用存储空间
- 数据库容量快速膨胀

#### 问题3：数据一致性风险
- 同步过程中可能出现数据不一致
- 事务处理复杂度高
- 回滚机制困难

### 1.3 关键字段说明

#### office 字段
- `office = 1`：表示是Magic服务商下的模型
- `office = 0`：表示是其他服务商（OpenAI、Claude等）下的模型
- Magic服务商的模型都来源于官方组织的各种服务商模型

#### modelParentId 字段
- 指向官方组织中对应的父模型ID
- 用于建立引用关系，实现配置继承

#### DisabledBy 字段逻辑
- `disabledBy = null`：正常激活状态
- `disabledBy = 'OFFICIAL'`：官方禁用，**用户完全不可见**
- `disabledBy = 'USER'`：用户禁用，**用户可见但处于禁用状态**

## 2. 优化方案设计

### 2.1 核心思想

#### 引用模式替代复制模式
- **当前**：每个组织在Magic服务商下保存完整的模型副本
- **优化后**：组织直接引用官方组织的模型，只在需要时创建自定义记录

#### 按需创建原则
- **当前**：预先同步所有模型到组织的Magic服务商下
- **优化后**：只有当组织修改模型状态时才创建记录

#### 查询时合并策略
- **当前**：直接查询组织Magic服务商下的模型
- **优化后**：查询官方组织所有模型 + 组织Magic服务商下的模型，运行时合并

### 2.2 数据流转图

```
官方组织 (official):
├── OpenAI服务商
│   ├── gpt-4 (office=0, modelParentId=0)
│   └── gpt-3.5 (office=0, modelParentId=0)
├── Claude服务商
│   └── claude-3 (office=0, modelParentId=0)
└── 无Magic服务商

业务组织A (org-a):
└── Magic服务商
    ├── 初始为空，不预先同步
    └── 用户修改状态后才创建:
        ├── gpt-4-custom (office=1, modelParentId=官方gpt-4的ID, status=USER禁用)
        └── claude-3-custom (office=1, modelParentId=官方claude-3的ID, status=激活)
```

### 2.3 核心机制

#### 2.3.1 查询合并逻辑
```php
public function getModelsForOrganization(string $organizationCode): array
{
    // 1. 查询官方组织的所有模型（排除官方禁用的）
    $officialModels = $this->getOfficialActiveModels();
    
    // 2. 查询当前组织Magic服务商下的模型
    $organizationModels = $this->getOrganizationMagicModels($organizationCode);
    
    // 3. 合并：如果模型重复，使用组织的记录
    return $this->mergeModels($officialModels, $organizationModels);
}
```

#### 2.3.2 配置继承机制
- **组织模型可修改字段**：`status`、`disabledBy`
- **继承父模型字段**：`name`、`icon`、`config`、`description`、`sort`、`translate`等所有其他字段

#### 2.3.3 状态优先级
1. **官方禁用**（`disabledBy = 'OFFICIAL'`）：完全不可见，无法创建组织记录
2. **组织自定义**：如果组织有自定义记录，使用组织的状态
3. **默认继承**：如果组织无自定义记录，使用官方模型状态

## 3. 详细实现代码

### 3.1 查询服务改造

#### ModelQueryService
```php
<?php

declare(strict_types=1);

namespace App\Domain\ModelAdmin\Service;

use App\Domain\ModelAdmin\Entity\ServiceProviderModelsEntity;
use App\Domain\ModelAdmin\Repository\Persistence\ServiceProviderModelsRepository;
use App\Domain\ModelAdmin\Constant\ServiceProviderCategory;
use App\Domain\ModelAdmin\Constant\ServiceProviderCode;
use App\Domain\ModelAdmin\Constant\Status;
use App\Domain\ModelAdmin\Constant\DisabledByType;

class ModelQueryService
{
    public function __construct(
        private ServiceProviderModelsRepository $modelsRepository,
    ) {}

    /**
     * 获取组织的模型列表（合并官方模型和组织自定义）
     */
    public function getModelsForOrganization(
        string $organizationCode, 
        ?ServiceProviderCategory $category = null
    ): array {
        
        // 1. 获取官方组织的所有激活模型（排除官方禁用）
        $officialModels = $this->getOfficialActiveModels($category);
        
        // 2. 获取当前组织Magic服务商下的模型
        $organizationModels = $this->getOrganizationMagicModels($organizationCode, $category);
        
        // 3. 合并模型：重复时使用组织的记录
        return $this->mergeModels($officialModels, $organizationModels);
    }

    /**
     * 获取官方组织的激活模型（排除官方禁用）
     */
    private function getOfficialActiveModels(?ServiceProviderCategory $category = null): array
    {
        $officialOrganization = config('service_provider.office_organization');
        
        $conditions = [
            'organization_code' => $officialOrganization,
        ];
        
        if ($category) {
            $conditions['category'] = $category->value;
        }
        
        $models = $this->modelsRepository->findBy($conditions);
        
        // 过滤掉官方禁用的模型
        return array_filter($models, function($model) {
            return $model->getDisabledBy() !== DisabledByType::OFFICIAL->value;
        });
    }

    /**
     * 获取组织Magic服务商下的模型
     */
    private function getOrganizationMagicModels(string $organizationCode, ?ServiceProviderCategory $category = null): array
    {
        $conditions = [
            'organization_code' => $organizationCode,
            'is_office' => 1, // Magic服务商下的模型
        ];
        
        if ($category) {
            $conditions['category'] = $category->value;
        }
        
        return $this->modelsRepository->findBy($conditions);
    }

    /**
     * 合并官方模型和组织模型
     */
    private function mergeModels(array $officialModels, array $organizationModels): array
    {
        // 将组织模型按 modelParentId 分组
        $orgModelMap = [];
        foreach ($organizationModels as $orgModel) {
            if ($orgModel->getModelParentId() > 0) {
                $orgModelMap[$orgModel->getModelParentId()] = $orgModel;
            }
        }
        
        $result = [];
        
        // 处理官方模型
        foreach ($officialModels as $officialModel) {
            if (isset($orgModelMap[$officialModel->getId()])) {
                // 如果组织有自定义记录，使用组织记录但继承官方配置
                $merged = $this->mergeOfficialWithOrganization($officialModel, $orgModelMap[$officialModel->getId()]);
                $result[] = $merged;
                unset($orgModelMap[$officialModel->getId()]);
            } else {
                // 如果组织没有自定义记录，直接使用官方模型（但调整组织信息）
                $merged = $this->adaptOfficialModelForOrganization($officialModel, $organizationModels[0]->getOrganizationCode() ?? '');
                $result[] = $merged;
            }
        }
        
        // 添加组织独有的模型（理论上不应该存在，但保险起见）
        foreach ($orgModelMap as $remainingModel) {
            $result[] = $remainingModel;
        }
        
        return $result;
    }

    /**
     * 合并官方模型和组织自定义记录
     */
    private function mergeOfficialWithOrganization(
        ServiceProviderModelsEntity $officialModel,
        ServiceProviderModelsEntity $organizationModel
    ): ServiceProviderModelsEntity {
        
        // 使用组织模型作为基础（保留ID、组织信息、状态）
        $merged = clone $organizationModel;
        
        // 继承官方模型的所有配置字段
        $merged->setName($officialModel->getName());
        $merged->setModelVersion($officialModel->getModelVersion());
        $merged->setModelId($officialModel->getModelId());
        $merged->setIcon($officialModel->getIcon());
        $merged->setConfig($officialModel->getConfig());
        $merged->setDescription($officialModel->getDescription());
        $merged->setSort($officialModel->getSort());
        $merged->setTranslate($officialModel->getTranslate());
        $merged->setVisibleOrganizations($officialModel->getVisibleOrganizations());
        $merged->setVisibleApplications($officialModel->getVisibleApplications());
        
        // 保持组织的状态字段不变
        // $merged->setStatus() - 保持组织设置
        // $merged->setDisabledBy() - 保持组织设置
        // $merged->setSuperMagicDisplayState() - 保持组织设置
        
        return $merged;
    }

    /**
     * 将官方模型适配为组织模型格式
     */
    private function adaptOfficialModelForOrganization(
        ServiceProviderModelsEntity $officialModel,
        string $organizationCode
    ): ServiceProviderModelsEntity {
        
        $adapted = clone $officialModel;
        
        // 调整组织信息
        $adapted->setOrganizationCode($organizationCode);
        $adapted->setModelParentId($officialModel->getId());
        $adapted->setIsOffice(true); // 标记为Magic服务商下的模型
        
        // ID置空，表示这是虚拟的合并记录
        $adapted->setId(null);
        
        return $adapted;
    }
}
```

### 3.2 状态更新服务改造

#### OrganizationModelService
```php
<?php

declare(strict_types=1);

namespace App\Domain\ModelAdmin\Service;

use App\Domain\ModelAdmin\Entity\ServiceProviderModelsEntity;
use App\Domain\ModelAdmin\Repository\Persistence\ServiceProviderModelsRepository;
use App\Domain\ModelAdmin\Repository\Persistence\ServiceProviderConfigRepository;
use App\Domain\ModelAdmin\Constant\Status;
use App\Domain\ModelAdmin\Constant\DisabledByType;
use App\Domain\ModelAdmin\Constant\ServiceProviderCode;
use App\ErrorCode\ServiceProviderErrorCode;
use App\Infrastructure\Core\Exception\ExceptionBuilder;

class OrganizationModelService
{
    public function __construct(
        private ServiceProviderModelsRepository $modelsRepository,
        private ServiceProviderConfigRepository $configRepository,
    ) {}

    /**
     * 更新组织模型状态（按需创建模型记录）
     */
    public function updateModelStatus(
        string $organizationCode,
        string $modelId, // 这是官方模型的 model_id 字段
        Status $status
    ): ServiceProviderModelsEntity {
        
        // 1. 找到官方组织的对应模型
        $officialModel = $this->findOfficialModelByModelId($modelId);
        if (!$officialModel) {
            ExceptionBuilder::throw(ServiceProviderErrorCode::ModelNotFound, '官方模型不存在');
        }
        
        // 2. 检查官方模型是否被官方禁用
        if ($officialModel->getDisabledBy() === DisabledByType::OFFICIAL->value) {
            ExceptionBuilder::throw(ServiceProviderErrorCode::ModelOfficiallyDisabled, '模型已被官方禁用，无法修改');
        }
        
        // 3. 查找或创建组织的模型记录
        $organizationModel = $this->findOrCreateOrganizationModel($organizationCode, $officialModel);
        
        // 4. 更新状态
        $organizationModel->setStatus($status->value);
        $organizationModel->setDisabledBy($status === Status::DISABLE ? DisabledByType::USER->value : null);
        
        // 5. 保存
        return $this->modelsRepository->save($organizationModel);
    }

    /**
     * 根据 model_id 找到官方模型
     */
    private function findOfficialModelByModelId(string $modelId): ?ServiceProviderModelsEntity
    {
        $officialOrganization = config('service_provider.office_organization');
        
        return $this->modelsRepository->findOneBy([
            'organization_code' => $officialOrganization,
            'model_id' => $modelId,
        ]);
    }

    /**
     * 查找或创建组织的模型记录
     */
    private function findOrCreateOrganizationModel(
        string $organizationCode,
        ServiceProviderModelsEntity $officialModel
    ): ServiceProviderModelsEntity {
        
        // 先尝试查找现有记录
        $existing = $this->modelsRepository->findOneBy([
            'organization_code' => $organizationCode,
            'model_parent_id' => $officialModel->getId(),
            'is_office' => 1,
        ]);
        
        if ($existing) {
            return $existing;
        }
        
        // 创建新的组织模型记录
        return $this->createOrganizationModel($organizationCode, $officialModel);
    }

    /**
     * 创建组织模型记录
     */
    private function createOrganizationModel(
        string $organizationCode,
        ServiceProviderModelsEntity $officialModel
    ): ServiceProviderModelsEntity {
        
        // 获取组织的Magic服务商配置ID
        $magicConfigId = $this->getMagicServiceProviderConfigId($organizationCode);
        
        // 创建新模型记录
        $organizationModel = new ServiceProviderModelsEntity();
        $organizationModel->setOrganizationCode($organizationCode);
        $organizationModel->setServiceProviderConfigId($magicConfigId);
        $organizationModel->setModelParentId($officialModel->getId());
        $organizationModel->setIsOffice(true); // Magic服务商下的模型
        
        // 继承官方模型的基本信息（这些字段在查询时会被覆盖，但为了数据完整性还是设置）
        $organizationModel->setName($officialModel->getName());
        $organizationModel->setModelVersion($officialModel->getModelVersion());
        $organizationModel->setModelId($officialModel->getModelId());
        $organizationModel->setIcon($officialModel->getIcon());
        $organizationModel->setCategory($officialModel->getCategory());
        $organizationModel->setConfig($officialModel->getConfig());
        $organizationModel->setModelType($officialModel->getModelType());
        $organizationModel->setDescription($officialModel->getDescription());
        $organizationModel->setSort($officialModel->getSort());
        $organizationModel->setTranslate($officialModel->getTranslate());
        
        // 初始状态继承官方模型
        $organizationModel->setStatus($officialModel->getStatus());
        $organizationModel->setDisabledBy($officialModel->getDisabledBy());
        
        return $organizationModel;
    }

    /**
     * 获取组织的Magic服务商配置ID
     */
    private function getMagicServiceProviderConfigId(string $organizationCode): int
    {
        $magicConfig = $this->configRepository->findOneBy([
            'organization_code' => $organizationCode,
            'provider_code' => ServiceProviderCode::Magic->value,
        ]);
        
        if (!$magicConfig) {
            ExceptionBuilder::throw(ServiceProviderErrorCode::MagicProviderNotFound, '组织Magic服务商配置不存在');
        }
        
        return $magicConfig->getId();
    }
}
```

### 3.3 应用层改造

#### ServiceProviderAppService 核心方法改造
```php
<?php

declare(strict_types=1);

namespace App\Application\ModelAdmin\Service;

use App\Domain\ModelAdmin\Service\ModelQueryService;
use App\Domain\ModelAdmin\Service\OrganizationModelService;
use App\Domain\ModelAdmin\Service\ServiceProviderDomainService;

class ServiceProviderAppService
{
    public function __construct(
        protected ServiceProviderDomainService $serviceProviderDomainService,
        protected ModelQueryService $modelQueryService,
        protected OrganizationModelService $organizationModelService,
        // ... 其他依赖保持不变
    ) {}

    /**
     * 获取服务商列表 - 使用新的查询逻辑
     */
    public function getServiceProviders(MagicUserAuthorization $authenticatable, ?ServiceProviderCategory $serviceProviderCategory): array
    {
        $organizationCode = $authenticatable->getOrganizationCode();

        // 使用新的模型查询服务
        $models = $this->modelQueryService->getModelsForOrganization($organizationCode, $serviceProviderCategory);
        
        // 获取服务商配置（只需要初始化配置，不需要初始化模型）
        $serviceProviderConfigs = $this->serviceProviderDomainService->getServiceProviderConfigs($organizationCode, $serviceProviderCategory);
        
        // 如果没有服务商配置，只初始化配置（不包含模型）
        if (empty($serviceProviderConfigs)) {
            $this->serviceProviderDomainService->initOrganizationServiceProviderConfigs($organizationCode, $serviceProviderCategory);
            $serviceProviderConfigs = $this->serviceProviderDomainService->getServiceProviderConfigs($organizationCode, $serviceProviderCategory);
        }
        
        // 组装服务商配置和模型
        $serviceProviderConfigDTOS = $this->buildServiceProviderConfigDTOsWithModels($serviceProviderConfigs, $models);
        
        // 后续处理逻辑保持不变
        foreach ($serviceProviderConfigDTOS as $serviceProviderConfigDTO) {
            $this->maskSensitiveConfigInfo($serviceProviderConfigDTO);
        }
        
        $this->processServiceProviderConfigIcons($serviceProviderConfigDTOS, $organizationCode);
        
        // 过滤官方组织的 Magic 服务商
        $officeOrganization = config('service_provider.office_organization');
        if ($authenticatable->getOrganizationCode() === $officeOrganization) {
            $serviceProviderConfigDTOS = array_filter($serviceProviderConfigDTOS, function ($serviceProviderConfigDTO) {
                return ServiceProviderCode::from($serviceProviderConfigDTO->getProviderCode()) !== ServiceProviderCode::Magic;
            });
            $serviceProviderConfigDTOS = array_values($serviceProviderConfigDTOS);
        }
        
        return $serviceProviderConfigDTOS;
    }

    /**
     * 更新模型状态 - 使用新的按需创建逻辑
     */
    public function updateModelStatus(string $modelId, int $status, string $organizationCode): void
    {
        $officeOrganization = config('service_provider.office_organization');
        
        if ($organizationCode === $officeOrganization) {
            // 官方组织直接更新官方模型，不影响其他组织
            $this->serviceProviderDomainService->updateOfficialModelStatus($modelId, Status::from($status));
        } else {
            // 非官方组织使用按需创建逻辑
            $this->organizationModelService->updateModelStatus(
                $organizationCode,
                $modelId,
                Status::from($status)
            );
        }
    }

    /**
     * 构建服务商配置DTO列表，包含模型信息
     */
    private function buildServiceProviderConfigDTOsWithModels(array $serviceProviderConfigs, array $models): array
    {
        // 将模型按服务商配置ID分组
        $modelsByConfigId = [];
        foreach ($models as $model) {
            $configId = $model->getServiceProviderConfigId();
            if (!isset($modelsByConfigId[$configId])) {
                $modelsByConfigId[$configId] = [];
            }
            $modelsByConfigId[$configId][] = $model;
        }
        
        // 为每个服务商配置设置对应的模型列表
        $result = [];
        foreach ($serviceProviderConfigs as $config) {
            $configModels = $modelsByConfigId[$config->getId()] ?? [];
            
            // 应用现有的过滤逻辑
            if ($this->isOfficialServiceProvider($config)) {
                $filteredModels = array_filter($configModels, function ($model) {
                    return $model->getStatus() === Status::ACTIVE->value
                        || ($model->getStatus() === Status::DISABLE->value && $model->getDisabledBy() === DisabledByType::USER->value);
                });
                $configModels = array_values($filteredModels);
            }
            
            $configDTO = new ServiceProviderConfigDTO($config->toArray());
            $configDTO->setModels($configModels);
            $result[] = $configDTO;
        }
        
        return $result;
    }
}
```

```php
<?php

declare(strict_types=1);

class ServiceProviderDomainService

    {
        $lockKey = "service_provider:init:{$organizationCode}";
        
        try {
            // 只初始化服务商配置，不同步模型

            // 释放锁
        }

        // 只确保Magic服务商配置存在

     * 初始化VLM服务商配置
     */
    private function initVLMServiceProviderConfigs(string $organizationCode): void
    {
        // VLM服务商配置的初始化逻辑保持不变

    }

    /**
     * 确保Magic服务商配置存在

    private function ensureMagicServiceProviderConfig(string $organizationCode): void

        $existing = $this->serviceProviderConfigRepository->findOneBy([
```

### 4.2 清理旧数据

#### 数据清理脚本
```php
<?php

declare(strict_types=1);

namespace App\Command\Cleanup;

use Hyperf\Command\Command as HyperfCommand;
use Hyperf\Command\Annotation\Command;

#[Command]
class CleanupRedundantModelsCommand extends HyperfCommand
{
    protected ?string $signature = 'model:cleanup-redundant {--dry-run : 只显示将要删除的数据}';

    protected string $description = '清理冗余的组织模型数据';

    public function handle()
    {
        $dryRun = $this->option('dry-run');
        $officeOrganization = config('service_provider.office_organization');
        
        // 查找所有非官方组织的Magic服务商下的模型
        $redundantModels = $this->modelsRepository->findBy([
            'is_office' => 1, // Magic服务商下的模型
            ['organization_code', '!=', $officeOrganization],
        ]);
        
        $this->info("发现冗余模型: " . count($redundantModels));
        
        if ($dryRun) {
            $this->showCleanupPreview($redundantModels);
            return;
        }
        
        // 执行清理
        $this->performCleanup($redundantModels);
        
        $this->info('清理完成!');
    }

    private function showCleanupPreview(array $models): void
    {
        $this->info("\n=== 清理预览 ===");
        
        $groupedByOrg = [];
        foreach ($models as $model) {
            $orgCode = $model->getOrganizationCode();
            if (!isset($groupedByOrg[$orgCode])) {
                $groupedByOrg[$orgCode] = 0;
            }
            $groupedByOrg[$orgCode]++;
        }
        
        foreach ($groupedByOrg as $orgCode => $count) {
            $this->info("组织 {$orgCode}: {$count} 个模型");
        }
    }

    private function performCleanup(array $models): void
    {
        $deleted = 0;
        $kept = 0;
        
        foreach ($models as $model) {
            // 只删除与官方模型完全相同的记录（没有自定义状态）
            if ($this->shouldKeepModel($model)) {
                $kept++;
                continue;
            }
            
            $this->modelsRepository->delete($model);
            $deleted++;
        }
        
        $this->info("删除: {$deleted}, 保留: {$kept}");
    }

    private function shouldKeepModel($model): bool
    {
        // 如果模型有自定义状态，应该保留
        if ($model->getDisabledBy() === DisabledByType::USER->value) {
            return true;
        }
        
        if ($model->getSuperMagicDisplayState() !== 0) {
            return true;
        }
        
        // 其他自定义判断逻辑...
        
        return false;
    }
}
```
